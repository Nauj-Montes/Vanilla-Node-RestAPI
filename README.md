# Vanilla Node REST API

```
# Routes
GET      /api/products
POST     /api/products
GET      /api/products/:id
PUT      /api/products/:id
DELETE   /api/products/:id

```

## Usage

```
# Install dependencies
npm install
yarn install

# Run in develpment
npm run dev
yarn run dev

# Use a client like POSTMAN
In it, you can get, post, get just one, update, and delete a product.
```
